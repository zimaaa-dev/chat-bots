# Веб-приложение для опросов по Excel прайс-листам

Это веб-приложение преобразовано из Telegram бота и предоставляет удобный интерфейс для проведения опросов на основе данных из Excel файлов.

## Установка и запуск

1. **Активируйте виртуальное окружение:**
   ```bash
   venv\Scripts\activate  # Windows
   source venv/bin/activate  # macOS/Linux
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements_web.txt
   ```

3. **Запустите приложение:**
   ```bash
   python web_app.py
   ```

4. **Откройте в браузере:**
   Перейдите по адресу: http://localhost:5000

## Структура проекта

- `web_app.py` - основное веб-приложение на Flask
- `templates/` - HTML шаблоны
  - `base.html` - базовый шаблон
  - `index.html` - главная страница
  - `question.html` - страница вопроса
  - `finish.html` - страница завершения
- `files_excel/` - папка с Excel прайс-листами
- `config.yaml` - конфигурация приложения
- `requirements_web.txt` - зависимости для веб-приложения

## Использование

1. На главной странице выберите прайс-лист
2. Нажмите "НАЧАТЬ" для начала опроса
3. Отвечайте на вопросы последовательно
4. Результаты автоматически сохраняются в Excel файл

## Особенности

- Поддержка различных типов вопросов (да/нет, множественный выбор, числа, даты, текст)
- Прогресс-бар для отслеживания процесса
- Адаптивный дизайн для мобильных устройств
- Автоматическое сохранение результатов

## Настройка

Для изменения настроек откройте файл `config.yaml`:

```yaml
FIRST_MESSAGE: 'Добро пожаловать!'
FILENAME_MESSAGE: 'Введите название файла'
FILENAME_ID_MESSAGE: 'Результаты сохранены'
REQUEST_FILENAME: true
REQUEST_USER: false
TYPE_BUTTONS: 'reply'
```
